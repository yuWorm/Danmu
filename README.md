#### 基本使用

-将插件上传到maccms的addons目录下，然后登陆后台，进入应用市场，启用即可。
-启用后刷新后台页面，即可看见应用菜单下有了弹幕管理选项。
-如禁用插件出现有冲突文件请点击弹幕管理的删除冲突文件按钮。
-启用插件后，会自动在播放器列表中添加弹幕播放器，记得清理缓存。
